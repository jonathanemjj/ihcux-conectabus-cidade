# 📱 Sobre o Projeto

O **ConectaBus ** é um protótipo de aplicativo desenvolvido para facilitar a vida de usuários do transporte público urbano, oferecendo informações rápidas e acessíveis sobre ônibus, linhas, horários e localização em tempo real.

O foco principal do projeto é garantir uma experiência simples, rápida e intuitiva, especialmente para pessoas em movimento ou com apenas uma mão disponível para utilizar o celular.

---

# 🌎 Contexto de Uso

O aplicativo foi pensado para situações reais do cotidiano urbano, como:

- Usuário correndo para não perder o ônibus.
- Pessoa segurando mochila, sacola ou bolsa enquanto utiliza o celular.
- Usuários em pontos movimentados e com pouco tempo para navegar.
- Pessoas tentando encontrar rapidamente a melhor linha disponível.
- Usuários utilizando o app em pé dentro do ônibus.

Por isso, o design prioriza:
- navegação simples,
- poucos toques,
- informações objetivas,
- botões grandes e acessíveis ao polegar.

---

# 🎯 Decisões de UX

Durante o desenvolvimento do protótipo, algumas decisões foram tomadas para melhorar a experiência do usuário:

## 🔹 Informações Prioritárias
Foram destacadas:
- número da linha,
- tempo estimado de chegada,
- destino do ônibus,
- botão de rota rápida.

Essas informações possuem maior destaque visual porque representam o que o usuário mais precisa em momentos de pressa.

---

## 🔹 Uso com Apenas um Polegar
Os principais botões foram posicionados na parte inferior da tela, facilitando o alcance com apenas uma mão.

Além disso:
- evitamos menus complexos,
- reduzimos excesso de texto,
- utilizamos poucos elementos por tela.

---

## 🔹 Fluxo Simples
O usuário consegue:
1. abrir o aplicativo,
2. pesquisar a linha,
3. visualizar o horário,
4. acessar a rota,

com poucos passos e baixa carga cognitiva.

---

# ♿ Acessibilidade

O protótipo também foi pensado para ajudar pessoas com mobilidade reduzida ou em situações de pressa.

## Recursos aplicados:
- Botões grandes e fáceis de tocar.
- Alto contraste entre texto e fundo.
- Informações principais em destaque.
- Navegação simplificada.
- Ícones intuitivos para facilitar compreensão rápida.
- Menor quantidade possível de etapas para completar ações.

Essas decisões ajudam:
- idosos,
- pessoas com dificuldades motoras,
- usuários distraídos,
- usuários em movimento,
- pessoas utilizando apenas uma mão.
